Python 数据结构：优先队列

Posted on 2017-08-26 | In 数据结构与算法 |

优先队列是一种特殊的队列类型，其中具有最高优先级的项会最先弹出，相同优先级的项按照 FIFO 的顺序弹出。

优先队列的主要方法

方法 作用
__len__(s) 返回优先队列中项的数目
__str__(s) 返回优先队列的字符串表示
__iter__() 从前到后依次访问优先队列中的每一项
__add__(s) 将两个优先队列合并
__eq__(s) 如果两个优先队列对应位置的项相等，两个优先队列相等
is_empty() 优先队列为空返回 False，否则返回 True
clear() 将优先队列清空
peak() 返回优先队列优先级最高项
pop() 删除优先队列优先级最高项并返回该项
add(item) 将元素添加优先队列中

优先队列的单链表实现

基于队列结构，可以实现一个简单的优先队列。

节点类：

定义队列类：

在 __init__() 中定义一个 size 用于表示当前队列中项的个数。实例变量 last 作为一个指针，用于指向队列的末尾项。

__len__(s)

优先队列类已经实现了一个实例变量 size ，只需返回这个实例变量即可。

__str__(s)

返回优先队列的字符串表示，将优先队列中个项的值封装成一个列表形式返回，优先队列较高的项在前。

__iter__()

优先级从高到低，依次访问各项，返回各项的值。

__add__(s)

两个优先队列相加。

__eq__(s)

与 == 相同，比较优先两个队列中项是否相等。

is_empty()

判断优先队列是否为空。

clear()

将优先队列全部清空

peak()

返回优先队列优先级最高项但不删除。

add(item)

将元素添加优先队列中。

pop()

返回优先队列优先级最高项并删除。当优先队列只有唯一项并被删除后，队列末尾的外部指针 last 与头部指针 head 都指向 None。
