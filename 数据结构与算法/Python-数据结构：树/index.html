Python 数据结构：二叉搜索树

Posted on 2017-10-02 | In 数据结构与算法 |

树是一种抽象数据类型，用来模拟具有树状结构性质的数据集合。

树有两个主要的特征：

  1. 每个项都有许多子节点，
  2. 除了根节点，其它项都有一个父节点。

树的深度和高度

树的深度又称为层级，一个节点的深度或层级等于将其连接到根节点的路径的长度。所以，根节点的深度和层级为0，根节点的子节点的层级为1。

一个空树的深度或层级为 -1。

树的高度表示树中最长路径的长度，即叶子节点的最大层级数。

二叉树

普通的树中节点的子结点可以有很多个，而二叉树中节点的子结点最多有两个，其每一个子树都是一个二叉树。上面图中的树就是一个二叉树。

二叉树分类

  1. 不平衡二叉树：存在子结点只有一个的节点的树。

  2. 完美平衡二叉树：最后一层的叶子节点都靠右对齐；

  3. 完全二叉树：最后一层的叶子节点都靠左对齐；

  4. 满二叉树：除最后一层无任何子节点外，每一层上的所有结点都有两个子结点二叉树。

二叉树两种应用

  1. 二叉搜索树

二叉搜索树

二叉搜索树是一棵二叉树，其中每个节点的值都大于其左子树中任意节点的值、小于其右子树中任意节点的值。

二叉搜索树的主要方法

方法 作用
__len__(s) 返回二叉搜索树中节点的数目
__str__(s) 返回二叉搜索树的字符串表示
__iter__() 迭代二叉搜索树的每一个节点
__add__(s) 将两个二叉搜索树合并
__eq__(s) 如果两个二叉搜索树的所有节点对应相等，两个树相等
is_empty() 二叉搜索树为空返回 False，否则返回 True
clear() 将二叉搜索树清空
搜索 搜索目标项在二叉搜索树中是否存在
遍历 遍历二叉搜索树搜索节点
插入 在二叉搜索树适当位置插入目标项
获取最大项和最小项 获取二叉搜索树的最大项和最小项
删除 删除二叉搜索树中的某一个节点

树节点和二叉搜索树初始化

一个树节点包含自身的值和左右子树的链接。二叉搜索树类中只要定义一个指向搜索树根节点的链接就可以包含整棵树。

len(s)

二叉搜索树类已经实现了一个实例变量 size ，只需返回这个实例变量即可。

__str__(s)

二叉搜索树字符串表示，将二叉搜索树逆时针旋转 90 度，并在内部节点之间添加 “|”。

__iter__()

借助于列表遍历节点的同时将其左右子树的节点添加到待遍历的列表中。

__add__(s)

两个二叉搜索树相加。

__eq__(s)

与 == 相同，比较两个二叉搜索树中所有节点是否相等。

is_empty()

判断二叉搜索树是否为空。

clear()

将二叉搜索树清空。

搜索

在二叉搜索树中搜索目标上。如果存在, 返回第一个匹配的项，否则返回 None。

遍历

遍历一个二叉搜索树有两种方式，即深度优先遍历和广度优先遍历；深度优先搜索又有三种方式，即前序遍历、中序遍历和后续遍历。

前序遍历

前序遍历先访问树的根节点，然后再分别遍历左子树和右子树。

中序遍历

中序遍历先遍历左子树，然后访问根节点，最后遍历右子树。

后序遍历

后序遍历先遍历左子树，然后遍历右子树，最后访问根节点。

广度优先遍历

二叉树的广度优先遍历又可以称为层序遍历，先从二叉搜索树的 0 层级开始，在每个层级按照从左到右的顺序访问节点。

插入

将一个项插入到二叉搜索树的适当位置，如果与二叉搜索树中某一项的值相等，插入到这一项的右子树中。

获取最大项和最小项

从二叉搜索树的根节点开始分别递归遍历右子树和左子树就可以找到最大项和最小项。

删除

删除操作中删除最大项或最小项是最简单的。

删除最大项或最小项

和获取最大项和最小项类似，只是多了删除操作。

删除某一项

删除某一项的操作要复杂一些，先要在二叉搜索树中找到要删除的节点，如果要删除的节点的左子树为空，就返回由子树，如果其右子树为空，就返回左子树；如果其左右子树都不为空，就在其右子树找一个最小的节点替换到当前节点。

找某个节点的右子树中的最小节点用到了一个辅助函数，返回并删除这个最小节点。
