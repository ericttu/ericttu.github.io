Mongdb 简介及 Python 连接

Posted on 2017-11-04 | In 数据库 |

MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

Centos 下安装与启动

官方文档

  1. 配置包管理：$ sudo vim /etc/yum.repos.d/mongodb-org-3.6.repo，添加以下内容：

  2. 安装 Mongodb 包：$ sudo yum install -y mongodb-org

    如果要安装指定版本： $ sudo yum install -y mongodb-org-3.6.3 mongodb-org-server-3.6.3 mongodb-org-shell-3.6.3 mongodb-org-mongos-3.6.3 mongodb-org-tools-3.6.3

  3. 在 /etc/yum.conf 中加入下面的配置防止意外升级： exclude=mongodb-org,mongodb-org-server,mongodb-org-shell,mongodb-org-mongos,mongodb-org-tools。

    mongodb 默认将数据文件防止 /var/lib/mongo 中，日志文件存放在 /var/log/mongodb 中。

    可以修改配置文件 /etc/mongod.conf 更改配置信息。

  4. Start MongoDB： $ sudo service mongod start。

    • 在 /var/log/mongodb/mongod.log 中出现 [initandlisten] waiting for connections on port \ 表示启动成功。
    • 端口号在/etc/mongod.conf中配置，默认为27017。

  5. 配置开机启动：$ sudo chkconfig mongod on

  6. Stop MongoDB： $ sudo service mongod stop
  7. Restart MongoDB： $ sudo service mongod restart

添加用户

  1. 开启用户认证：修改配置文件 /etc/mongod.conf，将 auth 设置为 true。

  2. 重启：$ sudo service mongod restart

  3. 添加管理员：进入 mongodb 命令行，选择 admin 数据库

    • role:root 表示最高权限，只在 admin 数据库中可用；
    • 第一个用户添加完成后，便需要认证才能继续添加其他用户。

  4. 用户认证：用户认证右两种方式

    1. 在客户端连接时就认证：$ mongo --port 27017 -u "root" -p "password" --authenticationDatabase "admin"；

    2. 客户端连接后再认证：$ mongo --port 27017

      输出 1 表示验证成功

  5. 添加普通用户：与创建管理员用户的过程类似

    用户在 a_db 数据库中创建，需要在 a_db 数据库验证用户身份。

限制 ip 访问

MongoDB 可以限制只允许某一特定IP来访问，修改配置文件 /etc/mongod.conf 中的 bind_ip。

在 Mongodb 服务器端添加了内网 ip，就可以通过外网 ip 访问。

命令行操作数据库

官方文档

首先选择一个数据库，如：use students。

新增数据

  1. mongdb 会自动在数据库中添加一个 ObjectId 类型的 _id 值，所以时间新增的数据为 {"_id" : ObjectId("5063114bd386d8fadbd6b004"), { "name" : "Jhon", "age" : 20 }。

    可以自己设定 _id 的值，如 {"_id" : 1, { "name" : "Jhon", "age" : 20 }。

  2. 插入多条需要将组数据以列表形式插入。

  3. 插入一条还可以用：insertOne()，插入多条还可以用：insertMany()。

查询数据

查询一条数据：db.students.findOne() 查询全部：db.students.find()

find() 可传两个参数：

  1. 第一个为查询的条件

    如：db.students.find({grade:{'$gte':60}}) 表示查询条件 grade >= 60，lte 表示小于等于。db.students.find({'$or': [{sex: 'male'}, {sex: 'female'}]}) 表示或条件。

  2. 第二个为指定显示内容「如：db.students.find({sex:"male"}, {name:1, age:true, _id:0})；1=true，0=false；_id 默认会显示，设定 0 或 false 使其不显示」
