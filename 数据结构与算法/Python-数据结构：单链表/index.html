Python 数据结构：单链表

Posted on 2017-08-11 | In 数据结构与算法 |

单链表是链表的一种，其链接方向是单向的，对链表的访问要通过从头部开始，依序往下读取。

  1. 在单链表中很容易访问一个项的后一项，但并不容易访问前一项。
  2. 链表在内存中的表示使用非连续性内存，即只需给定每个项的地址和链接，便能在内存中任一位置找到。

删除
