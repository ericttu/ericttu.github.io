Python 基础：字典

Posted on 2017-07-01 | In Python 基础

在 Python 中，字典是一系列键 — 值（key - value）对。每个键都与一个值相关联，你可以使用键来访问与之相关联的值。可将任何 Python 对象用作字典中的值。

全文采用 Python3 版本

字典的特点

  1. 查找速度快；
  2. 存储的 key - value 对是没有顺序的；
  3. 作为 key 的元素必须不可变。

更改字典的键 — 值对

  1. 添加键 — 值对
    添加键 — 值对，可依次指定字典名、用方括号括起的键和相关联的值。

  2. 修改字典的值
    修改字典中的值，可依次指定字典名、用方括号括起的键以及与该键相关联的新值。

  3. 删除键 — 值对
    可使用 del 语句彻底删除键 — 值对

遍历字典

  1. 遍历所有键 — 值对
    可使用 for 循环遍历字典。

    items() 方法返回一个键—值对列表。

  2. 遍历字典中的键或值
    可使用 key() 方法遍历字典中的键。

    .key() 可省略

    可使用 values() 方法遍历字典中的值

  3. 按顺序遍历遍历字典中的键或值
    要以特定的顺序返回元素，可在for循环中使用 sorted() 函数进行排序。

  4. set（集合）
    集合类似于列表，但每个元素都必须是独一无二的。可用来去除重复项。

嵌套

将一系列字典存储在列表中，或将列表作为值存储在字典中，称为嵌套。

  1. 列表中存储字典

  2. 字典中存储列表

  3. 字典中存储字典
