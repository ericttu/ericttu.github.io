篮球计分器实践

Posted on 2017-04-29 | In 移动开发 |

Google Study Jams 活动官网

目标

根据两个球队所得分数（1、2、3分）累计各队总分并展示，比赛结束后分数清零。

布局

这里使用 RelativeLayout，包含有：

  1. ImageView(背景图片）

  2. LinearLayout(A队、B队)

    • LinearLayout(A队)

      TextView(队名)
      TextView(分数)
      Button(+1分)
      Button(+2分)
      Button(+3分)

    • LinearLayout(A队)

      TextView(队名)
      TextView(分数)
      Button(+1分)
      Button(+2分)
      Button(+3分)

  3. button(重置按钮)

UI优化

操作栏和按钮颜色在styles.xml中统一设置。
colorPrimary将更改操作栏的颜色。colorButtonNormal将更改按钮的颜色。

注意：colorButtonNormal只能在搭载 API 22 及以上版本的手机上起到作用。如果手机版本更低，这些按钮将不会变成设置的颜色。
